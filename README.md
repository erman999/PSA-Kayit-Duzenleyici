# PSA Kayıt Düzenleyici

*PSA makinalarına ait üretim kayıtlarını gösterilen hedef klasörden okur ve hatalarını düzelterek masaüstüne oluşturduğu çıktı dosyasının içine atar.*

#### Kurulum

```
npm install
```


#### Çalıştırmak için

```
npm start
```


#### Paketlemek için

```
npm run-script build
```
