# PSA Kayıt Düzenleyici

*PSA makinalarına ait üretim kayıtlarını gösterilen hedef klasörden okur, hatalarını düzeltir ve masaüstüne hedef klasör ile aynı isimde bir çıktı üretir.*

#### Kurulum

```
npm install
```


#### Çalıştırmak için

```
npm start
```


#### Paketlemek için

```
npm run-script build
```
